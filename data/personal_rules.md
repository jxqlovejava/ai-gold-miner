## 补充规则（p系列，与15条军规互补）

| ID | 规则 |
|----|------|
| p001 | 重大数据前1-2天完成仓位调整，不赌数据方向 |
| p002 | 用条件单代替盯盘手动单，提前挂好避免盘中犹豫 |
| p003 | 减仓趁反弹出，不追求卖最高点——出不出比多卖几块重要100倍 |
| p004 | 单日波动>3%当天不操作，等次日 |
| p005 | ETF主力资金流向（当日）比CFTC（滞后8天）更及时 |
| p006 | 散户抄底+机构出货的反弹不可持续 |
| p007 | 浮亏超10%后决策质量骤降，提前动作不要等 |
| p008 | 每笔操作前问：如果空仓，会在这个价格买入吗？不会就减 |
| p009 | 买卖前先看机构/聪明钱/散户资金流向。短期上涨+散户抄底+机构出货=接飞刀，不进场 |

## 快速检查（每次交易前3问）

1. 规则驱动还是情绪驱动？
2. 48小时内有高影响事件吗？
3. 空仓状态下会在这个价格操作吗？
4. 机构资金在流入还是流出？
